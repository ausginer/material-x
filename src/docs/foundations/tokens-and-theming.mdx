# Tokens and Theming

Material X uses a token-driven pipeline and exposes stable public CSS variables
(`--md-*`) for customization.

## Theming Model

- System colors for Storybook docs are generated into `.storybook/theme.css`.
- Component styling is built from Material 3 token sets and adjusted through the
  local token processor.
- Private internal variables (`--_...`) are implementation details and should
  not be relied on by application code.

## Public Overrides

Override public custom properties on host elements or theme scopes:

```css
:root {
  --md-sys-color-primary: #0057d8;
}

mx-button {
  --md-button-container-height: 44px;
}
```
